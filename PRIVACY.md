# Jet Mia Privacy Policy

Jet Mia is designed for local file sharing. The app does not require an account, does not show ads, and does not intentionally collect personal data.

Files and messages are shared directly between devices on your local network or through the transfer mode you choose in the app. Jet Mia does not operate a separate tracking service for your transfers.

Some platform features, such as Android media permissions or desktop file pickers, may ask for access so the app can send or save files. Those permissions are used only for the feature you request.

For source code and issue reporting, visit https://github.com/SHNWAZX/jet-mia.
