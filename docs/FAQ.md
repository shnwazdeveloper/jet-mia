# FAQ

## Where should I ask questions?

Use GitHub Discussions for support questions:

https://github.com/SHNWAZX/jet-mia/discussions

## What kind of project is this?

Jet Mia is a LocalSend-based file sharing app. Check the README first for setup and project-specific notes.

## What should I include in a support question?

Include the app version or commit, your platform, the steps you followed, what you expected, what happened, and any relevant logs or screenshots.

## When should I open an issue instead?

Open an issue when you can describe a reproducible bug or a specific feature request. Use Discussions when you are asking how something works or need troubleshooting help.

## How do accepted answers work?

If someone answers your question in a GitHub Discussion and it solves the problem, use the accepted-answer control on that answer. This helps future users find the working solution quickly.
