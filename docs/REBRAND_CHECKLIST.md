# Rebrand Checklist

Use this checklist to verify Jet Mia branding changes.

## Visual identity

- App name appears consistently in launch screens, settings, and package metadata.
- Icons and splash assets match the current brand.
- Old product names do not appear in user-facing text.

## Functional checks

- File transfer flows still work after branding changes.
- Platform-specific package identifiers are correct.
- Release notes clearly separate upstream changes from local branding changes.
