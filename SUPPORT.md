# Support

Thanks for using Jet Mia. This repository uses GitHub Discussions for questions and community support.

## Ask a question

Open a new discussion here:

https://github.com/SHNWAZX/jet-mia/discussions

Use the Q&A category when you need help with setup, usage, builds, or troubleshooting. Include the version or commit you are using, your device or platform, the steps you tried, and the exact error message if there is one.

## Report a bug

Use GitHub Issues for reproducible bugs. A helpful report includes:

- What you expected to happen
- What happened instead
- Steps to reproduce the problem
- Screenshots, logs, or build output when available

## Accepted answers

When an answer solves your question, mark it as accepted in the discussion. That keeps the support area useful for everyone who finds the same question later.
